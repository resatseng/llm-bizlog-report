### 📊 **階段 2：效果驗證（2-4 週）**

#### 追蹤指標設計

**Output 1（推薦問項）的成效指標：**
```
補問商機轉換率 = 推進到下一階段的公司數 / 總補問公司數
目標：> 30%（正常商機轉換率約 20%）

追蹤欄位：
- company_id
- 補問前階段
- 補問後階段
- 是否成功補充資訊（L4/L5）
- 推進時間（天）
```

**Output 2（開發方案卡）的成效指標：**
```
需求驗證準確率 = 實際開發後獲客數 / 預估影響公司數
目標：> 50%

追蹤欄位：
- pain_type
- 預估 company_count
- 實際開發功能
- 實際獲客數
- ROI
```

**Output 3（機會卡）的成效指標：**
```
B+A 成交率 = 實際成交數 / 總 B+A 公司數
目標：A 階段 > 60%、B 階段 > 40%

追蹤欄位：
- company_id
- 跟進次數
- 成交金額
- 成交時間
- 卡點原因（如未成交）
```
