# 痛點分類結果（完整版）

## 概述
本階段將 123,893 筆 L1 痛點記錄進行精細分類，達成 **100% 分類率**（無「其他」類別），並生成多維度視覺化分析圖表。

## 分類方法
採用**混合式分類法**：
1. **關鍵字匹配**：189 個關鍵字涵蓋 12 大痛點類別
2. **規則推論**：基於痛點文本語義進行智能分類
3. **迭代優化**：分析未分類記錄，擴充關鍵字庫

## 關鍵成果

### 1. 數據簡化
- **原始組合**：89,710 個（cluster × pain_type）
- **簡化後**：83 個有效組合
- **壓縮率**：1,080 倍

### 2. 熱度計算公式
```
heat_score = ba_ratio × 0.4 + urgency_high_pct × 0.3 + scale_score × 0.2 + avg_confidence × 0.1
```

參數說明：
- `ba_ratio`: B+A 階段公司比例（商機成熟度）
- `urgency_high_pct`: 高緊迫度比例（時效性）
- `scale_score`: 規模分數（市場規模）
- `avg_confidence`: 平均信心度（分類可靠性）

### 3. 核心數據檔案

| 檔案名稱 | 說明 | 記錄數 |
|---------|------|--------|
| `pain_categories_extended.json` | 12 類別定義 + 189 關鍵字 | - |
| `pain_records_完整分類.csv` | 痛點明細（含分類） | 123,893 |
| `pain_heatmap_完整分類.csv` | 熱力圖數據（cluster × category） | 83 |
| `分類統計.csv` | 各類別統計摘要 | 12 |
