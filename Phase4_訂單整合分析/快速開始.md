## 📊 立即可用的洞察

### 洞察 1：熱圖是否準確？

**打開：`A1_heat_score_validation.csv`**

看兩個關鍵欄位：
- `heat_score`：Phase 3 計算的熱度分數
- `actual_conversion_rate`：實際成交率（2024-2026）

**期望結果：**
- 相關係數 > 0.3：模型有效 ✓
- 相關係數 < 0.1：模型需要調整 ⚠️

**如何使用：**
```
如果 heat_score 高但 actual_conversion_rate 低
→ 該痛點組合可能被高估，降低優先順序
→ 檢查是否有其他因素影響成交

如果 heat_score 低但 actual_conversion_rate 高
→ 該痛點組合被低估，值得關注
→ 考慮調整 heat_score 公式權重
```

### 洞察 3：最高 ROI 的痛點

**打開：`B3_pain_roi_ranking.csv`**

按 `roi_per_company` 降序排列：

```

→ 代表每觸及一家有此痛點的公司，平均能帶來多少產值
```

**為什麼要看 ROI？**

| 情況 | total_revenue | total_companies | ROI | 策略 |
|------|---------------|-----------------|-----|------|
| 痛點 A | 1 億 | 1,000 家 | 10 萬/家 | 大眾市場，廣撒網 |
| 痛點 B | 5,000 萬 | 50 家 | 100 萬/家 | 利基市場，精準打擊 |

**如何使用：**
```
高 ROI + 高總產值 → 第一優先（又大又肥）
高 ROI + 低總產值 → 利基市場（小而美）
低 ROI + 高總產值 → 大眾市場（薄利多銷）
低 ROI + 低總產值 → 低優先（考慮放棄）
```

### 行動 3：建立預測模型（選用）

使用 `D1_feature_matrix.csv`：

```python
import pandas as pd
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import train_test_split

# 載入特徵矩陣
df = pd.read_csv("results/D1_feature_matrix.csv")

# 準備特徵與目標
X = df[["cluster", "pain_type_encoded", "stage_encoded"]]
y = df["has_order"]

# 訓練模型
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)
model = RandomForestClassifier()
model.fit(X_train, y_train)

# 預測成交機率
df["predicted_prob"] = model.predict_proba(X)[:, 1]
```
