## 📞 遇到問題怎麼辦？

### 常見問題速查

**Q: 我的客戶不在任何輸出檔案中？**
→ 可能是：(1) 商機溫度太低（E/D/none）(2) 沒有 L1 痛點資料
→ 正常跟進即可，不影響既有流程

**Q: Output 2 的痛點太抽象，怎麼轉化成產品需求？**
→ 參考《使用指南.md》的「產品場景」章節
→ 開跨部門會議，邀業務解讀痛點細節

**Q: 數據多久更新一次？**
→ 建議每月重跑 Phase 3
→ 商機階段資料越新，熱圖越準確
